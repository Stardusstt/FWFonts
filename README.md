# WMfonts

Better fonts for users .

## WMfonts ( Magisk Modules for Android )

Full weight and macOS fonts patch for Android devices .

  ### Introduction
  
  Requires Android 9 or later .
  
  ### Support language
  
  * English
  * Chinese ( 繁 / 簡 )
  * Japanese


  ### Fonts
  
  * English : San Francisco Pro
  * Chinese : PingFang (蘋方)
  * Japanese : Hiragino MaruGothic (ヒラギノ丸ゴ)
